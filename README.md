#ixinyi-server

个人博客server端，使用 `node.js` + `express` + `leancloud` 开发， 在这里 ** [leancloud]（https://leancloud.cn）** 起到存储数据的作用，相当于  `mongodb`，但是在该项目中并没有使用  `LeanCloud` 已经封装好的 `REST-API` 地址:[https://leancloud.cn/docs/rest_api.html](https://leancloud.cn/docs/rest_api.html), 而是按照自己的需求进行封装相应的接口层。封装好的底层接口放在  `controlless` 下的  `leancloudDao.js` 中。
